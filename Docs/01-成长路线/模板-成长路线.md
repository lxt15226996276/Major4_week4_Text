## 一、五层进阶模型

```
试卷 L1 → P-L1 → P+1 → L1+ / 📌A / 🔷B
```

| 层级 | 说明 |
|------|------|
| **P-L1** | 个人及格 · ≥ 学员现有写法 |
| **P+1** | 本套 3～5 条提升 |
| **A v7.2** | 是什么 · 卷面为何 · 再挖一层 |
| **B** | FICM 八维 + 对照代码 · **链小灶** |
| **小灶** | `小灶：XXX` → [`小灶课堂/`](小灶课堂/) 深度讲义 |

## 四、AI 教练规则

| 指令 | 行为 |
|------|------|
| 进行下一步 | 读档案 + 读本步代码 · L1/P-L1/P+1 + **A + B** |
| 检查一下 | 双档验收 · 不改仓库 |
| 帮我改 | 授权后可写 Scripts |

**禁止**：v7.3/v7.4 式 A · 低于 P-L1 范本 · 代做整卷

## 四-B、Coach 自我迭代（v7.12 · §11.18）

| 环节 | 规则 |
|------|------|
| **触发** | 学员 **「迭代规范/自我进化/同意迭代 v7.x」**；或 Coach 攒够 **≥3 次同类失误** 先提案 |
| **流程** | 自检 → `Coach迭代提案_vX_待批准.md` → **学员同意** → 改规范/模板/档案 |
| **禁止** | 「进行下一步」中途升版 · 进行中 Exam retro 改 P-L1 |
| **踩坑库** | 档案 **§十一** · 与 §七 已掌握模式分工 |
