## 四、刷题执行标准（合并豆包 30/70 与 v6）

```
单次学习块（2～3h）
├── 10%  读考纲 / 教程 §考点 + 关联 U（豆包「理论速学」压缩版）
├── 70%  自己敲（Exam 小步 / Lab 分步）— Chat「进行下一步」
├── 15%  「检查一下」双档 L1 + P-L1
└── 5%   错题库 1 行 + U 表更新
```

**与豆包差异**：不是「先看完理论清单再刷题」，而是 **每套 Exam 教程已含考点表**，做到哪学到哪。
