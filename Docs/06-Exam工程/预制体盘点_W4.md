## 四、W4 各阶段 · 推荐引用组合

### Stage1_Login（D1～D2）

```
LoginScene     → Exam01/LoginCanvas 或 Exam02/LoginSceneUI
LoadingScene   → Exam06/LoadingCanvas（首选）
ServerScene    → Exam08/ServerCanvas + ServerItem（可选）
```

**Min 链**：Login → Loading → Main（3 场景 · Build 0→1→2）

### Stage2_Main（D3）

```
MainScene      → Exam01/MainCanvas（Unpack）+ Exam10/HPPanel 或 Exam06/GameSceneUI 人物区
```

**Min 验收**：左上人物信息 · 右下商城/技能入口钮

### Stage3_Inventory（D4）

```
背包           → Exam01/MainCanvas/BackpackPanel（已有 Scroll + Grid）
物品格         → Exam01/SkillItem 改图改字
商城           → Exam04/GameSceneUI 或 Exam06/GameSceneUI 商城区
```

**Min 验收**：动态 N 个物品 · 关闭返回主界面

### Stage4_Battle（D5）

```
主角           → Exam04/Player 或 Exam07/Cube
敌人           → Exam04/Enemy 或 Exam03/CubeEnemy
子弹/技能      → Exam03/Bullet
地面           → Exam03/Plane 或 Exam04/Plane
UI             → Exam08/GameSceneUI（技能钮 + 冷却）
```

**Min 验收**：WASD 移动 + 动画 · 至少 1 种攻击 · 1 个敌人交互

## 六、使用纪律（O1d · K1）

| 规则 | 说明 |
|------|------|
| **Glob 先于手搭** | 开步前 `Glob **/Login*.prefab` · 有则 Unpack |
| **禁止** | 仓库已有 LoadingCanvas 仍 UI→Canvas 手搭 |
| **引用方式** | 拖 Prefab 进 Stage 场景 · Unpack Completely · 删 Missing Script |
| **不改池内 Prefab** | 修改在 Stage 场景副本上进行 · 素材池保持干净 |
