## 六、启发总结（写进教练规则）

1. **案例驱动 ≠ 案例全做**：SiKi 用 genre 教技术；你用 **Exam/Lab/RPG** 教考纲 — 同构不同题池。  
2. **选修显式化**：Shader/性能/框架/网络在 SiKi 标【选修】→ 你的 **W3 后 / 阶段 02 后** 才开。  
3. **C# 永不毕业**：SiKi S1→S7 螺旋 → 档案 **D2 维** 每月复测，不回头刷 S1 小题。  
4. **网络阶段独立**：SiKi 第四阶段 90 课 → 你 **K6/U20 达标前不碰联机**。  
5. **学前必读先收尾**：12/20 → 补完再开长案例，避免心态分散。
