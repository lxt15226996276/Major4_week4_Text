## 一、Button · Sprite Swap 四态（Exam05 · 2026-06-24 · **你亲自踩完**）

> **场景**：`Assets/Images/003-taskother/btn_技能图标N_1～4` · 四扣血钮 · Transition = **Sprite Swap**

| # | 你看到的（症状） | 根因 | **禁止再犯** | **现行标准（v7.12.3）** |
|:-:|-----------------|------|--------------|-------------------------|
| **B1** | 悬停/按压 **几乎没手感**，只变亮不变大 | 四态 **只调亮度**，画布内图标同尺寸 | 禁止「四态=滤镜」 | Hover **+10%** · Press **-4%** 烘焙进 PNG |
| **B2** | 悬停 **圆环缺边、被切掉一块** | 在小画布（142px）内放大 → **PNG 边界裁像素** | 禁止在原图尺寸内放大 | 统一 **256×256** + 透明边距 |
| **B3** | 悬停 **反而比默认更小** | `min(0.828, 0.70)` → Hover **0.70 < Normal 0.72** | 禁止 `min()` 把 Hover cap 低于 Normal | **断言** `FILL_HOVER > FILL_NORMAL > FILL_PRESSED` |
| **B4** | 按压 **缩成一团、不合理** | Pressed **86%** 过度 | 禁止 Press **<95%** | Press **96%**（行业 95~98%） |

### 现行参数（脚本唯一真相 · 改比例只改这里）

| 参数 | 值 | 含义 |
|------|-----|------|
| `OUTPUT_SIZE` | **256** | 四态同尺寸 |
| `FILL_NORMAL` | **0.60** | 普通态占画布 |
| `RATIO_HOVER` | **1.10** | 悬停 +10% |
| `RATIO_PRESSED` | **0.96** | 按压 -4% |
| 顺序自检 | bbox 或 assert | **_2 > _1 > _3** |

**Unity 挂法**：Image **Simple** · 父级 **无 Mask** · Reimport 后重拖 `_1～_4`。
