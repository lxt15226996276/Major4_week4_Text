#### 战斗血条 `HpSlider`（若本步含 · **必填 §11.20**）

| 物体/字段 | 标准值 | 说明 |
|-----------|--------|------|
| Slider · Min / Max / Value | 0 / 100 / **100** | 开局满血 |
| Whole Numbers · Interactable | **开** · **关** | 整数 HP · 脚本驱动 |
| Handle Rect | **None** | 无手柄 |
| Fill Rect | **Fill** | Fill Area 子物体 |
| **Fill @ HP=0** | **Active 开** · 仅 `value=0` | **禁止**代码 SetActive(false) Fill |
| Background | 可见 · Raycast **关** | 0 血时空槽仍显示 |
| Fill · Image · Raycast | **关** | |

Hierarchy 参考：

```
HpSlider（或 HPSlider）
├── Background
├── Fill Area
│   └── Fill          ← Fill Rect 拖此 · 0 血时宽=0 · 仍 Active
└── Handle Slide Area（可删或留空 · Handle=None）
```

#### 战斗钮行 `AttackButtonRow`（若本步含 · **必填 §11.25 · 含 U2**）

| 物体/组件 | 标准值 | 说明 |
|-----------|--------|------|
| **BattlePanel** | Anchor **底边 Stretch 宽** · H **220** | 技能栏分区 |
| 容器 | **`AttackButtonRow`** | **直接**挂 BattlePanel · 禁中间 100×100 小盒 |
| **Content Size Fitter** ★ | Horizontal **Preferred Size** | 行宽随钮数 · **居中对称扩展** |
| **Horizontal Layout Group** | Spacing **24** · Force Expand **关** | 禁止 Spacing 乱填导致溢出 |
| **RectTransform** | Anchor 底边中 · **Pivot 0.5, 0.5** | U2 防单侧溢出 |
| 每个 Button | **Layout Element** Preferred **200×200** | 禁止手改 Pos |

Hierarchy 参考：

```
BattlePanel (底边 Stretch)
└── AttackButtonRow (HLG + CSF · Pivot 0.5)
    ├── BtnAttack20
    ├── BtnAttack30
    └── BtnAttack50
```

**踩坑 U1**：绝对 Pos 手摆 · **U2**：有 HLG 仍溢出 → 缺 CSF / Pivot / Spacing 过大 / Force Expand 开。
