## 门禁 B · G1 资产门禁（§11.13 · O1d · UI 大步强制）

> **根因**：Exam04 Loading 手搭 · Exam08 Login 手搭 — **发 Chat 前未 Glob Prefab**

### 内部必做（UI 大步 · 工具层 · 学员不可见也要做）

```
① Glob `Assets/Exams/**/Prefabs/*.prefab`（或 Grep 本步 UI 类型名）
② 写「Prefab 候选表」：路径 · 结论 ✅/❌ · 差量
③ 有 ✅ 候选 → Chat/讲义第一小节 = **X.0 体检 + Unpack** · 禁止「UI→Canvas 手搭」为首方案
④ 完全无候选 → 显式写「本步无同类 Prefab · 手搭」+ §11.26 全表
```
