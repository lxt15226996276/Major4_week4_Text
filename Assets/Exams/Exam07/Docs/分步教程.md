## 第 4 步：相机跟随 + **2D 混合树**移动动画（考点 3）

> 状态：**✅ 已验收（2026-07-12）** · `MasterBlendTree` 2D混合树 · `BindTarget` 动态相机

### 4.1 读题 · 本步拆两块

| 块 | 卷面 | 本步实现 |
|----|------|----------|
| A 动画 | 主角移动有正确行走姿态 | **2D Freeform Directional** · `VelocityX` + `VelocityZ` |
| B 相机 | 相机跟随主角 | `CameraFollow` · `LateUpdate` · 平滑跟随 |

### 4.2 动画资产（P0 · 全项目池）

`Player.prefab` 使用 **`Assets/Animator/MasterBlendTree.controller`** · 动画在 **`Assets/Animation/Master/`**

| 动画 | 用途 | 混合树位置 (X, Z) |
|------|------|:-----------------:|
| `Idle` | 站立 | **(0, 0)** |
| `Walk_F` | 前进 | **(0, 1)** |
| `Walk_B` | 后退 | **(0, -1)** |
| `Walk_L` | 左移 | **(-1, 0)** |
| `Walk_R` | 右移 | **(1, 0)** |

> P+1：可加 `Run_F/B/L/R` 做外圈 · 本步 L1 **Walk 四向 + Idle** 即可。

### 4.3 新建 `Exam07_Player.controller`（勿直接改全局 Master）

1. 复制 `Assets/Animator/Master.controller` → `Assets/Exams/Exam07/Animator/Exam07_Player.controller`  
2. **Parameters** 新增（或保留并改用）：
   - **`VelocityX`** · Float · 默认 0  
   - **`VelocityZ`** · Float · 默认 0  
3. Base Layer → 右键 → **Create State → From New Blend Tree** · 命名 **`Locomotion`**  
4. 双击 Blend Tree → Inspector：
   - **Blend Type** = **2D Freeform Directional**  
   - **Parameters** = `VelocityX` · `VelocityZ`  
5. 添 5 个 Motion · 按上表设 **Pos X / Pos Y**（= VelocityX / VelocityZ）  
6. **Locomotion** 设为 **默认状态** · 可 **禁用** 原 Idle↔Walk 的 `IsWalking` BOOL 过渡（避免打架）  
7. `Player.prefab` → Animator **Controller** 换为 **`Exam07_Player`**

### 4.4 改 `PlayerMove.cs`（挂 Player Prefab）

**删掉** `SetBool("IsWalking", …)` · 改为驱动 2D 混合树：

```csharp
// 思路：输入 → 本地速度向量 → 阻尼写入 VelocityX/Z → CC 位移 + 面朝移动方向
float horizontal = Input.GetAxisRaw("Horizontal");
float vertical = Input.GetAxisRaw("Vertical");
Vector3 direction = new Vector3(horizontal, 0f, vertical);

if (direction.sqrMagnitude > 0.01f)
{
    direction.Normalize();
    _controller.Move(direction * moveSpeed * Time.deltaTime);
    transform.rotation = Quaternion.LookRotation(direction);
}

// 2D 混合树：用输入轴（-1~1）驱动；SetFloat 第3参=阻尼时间
_animator.SetFloat("VelocityX", horizontal, 0.1f, Time.deltaTime);
_animator.SetFloat("VelocityZ", vertical, 0.1f, Time.deltaTime);
```

| 参数 | 含义 |
|------|------|
| `VelocityX` | 左右输入 · 混合 `Walk_L` / `Walk_R` |
| `VelocityZ` | 前后输入 · 混合 `Walk_F` / `Walk_B` |
| 阻尼 `0.1f` | 松键后平滑回 Idle · 避免动画瞬切 |

> 进阶：可用 `transform.InverseTransformDirection(worldVel)` 代替 raw 输入（角色已转身时更准）· P+1。

### 4.5 `CameraFollow` + `PlayerSpawner.BindTarget`（动态主角）

- `CameraFollow.BindTarget(Transform)` · **生成后**才记录 `localOffset`
- `PlayerSpawner.Start` → `Instantiate` → `cameraFollow.BindTarget(player.transform)`
- **禁止** Play 后手拖 `Player(Clone)`（W4d6）

- [ ] 相机 **跟随** Player 移动 · 不抖、不丢目标  

**P-L1**

### 📌 原理加餐（A）

- **2D 混合树**：用 **两个 Float** 当平面坐标 · 在 Idle/Walk 各向片段间 **加权混合** · 斜走 = 多向同时加权。  
- **Freeform Directional**：适合 **前后左右** 离散 walk 片 · 比 1D `Speed` 更能表达方向。  
- **LateUpdate**：相机在主角位移 **之后** 跟 · 减少一帧滞后抖动。

### 5.3 脚本 · `ShopController.cs`（挂 GameCanvas 或 `ShopManager`）

- 字段：`[SerializeField]` 拖 Button / GameObject / Slider / Text
- `Start`：注册 `onClick` · `ShopPanel.SetActive(false)`
- `BuyHP` / `BuyMP`：金币不足 → **`Debug.Log("购买失败")`** · 够则扣费 → 刷新 UI → **`Debug.Log("购买成功")`**

**购买方法关键行（卷面 L1 · 对照敲）**

```csharp
private void BuyHp()
{
    if (_gold < hpPotionCost)
    {
        Debug.Log("购买失败");   // 卷面 L1 · 金币不足
        return;
    }

    _gold -= hpPotionCost;
    sliderHp.value = Mathf.Min(sliderHp.maxValue, sliderHp.value + hpRestore);
    RefreshUI();
    Debug.Log("购买成功");
}

private void BuyMp()
{
    if (_gold < mpPotionCost)
    {
        Debug.Log("购买失败");
        return;
    }

    _gold -= mpPotionCost;
    sliderMp.value = Mathf.Min(sliderMp.maxValue, sliderMp.value + mpRestore);
    RefreshUI();
    Debug.Log("购买成功");
}
```
