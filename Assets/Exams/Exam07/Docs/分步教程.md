### 1.3 场景操作

| # | 操作 |
|:-:|------|
| 1 | 新建或打开 `Assets/Exams/Exam07/Scenes/Exam07_Game.unity`（可沿用 `Exam07_GameScene`） |
| 2 | 拖 **`GameCanvas.prefab`** 进 Hierarchy（已有则跳过） |
| 3 | **补 `EventSystem`**（缺则 UI 悬停/点击无效 · Sprite Swap 不动） |
| 4 | 确认 **Main Camera** · **Directional Light** |
| 5 | `Build Settings` → 加入本场景（单场景 Play 验收即可） |

**Hierarchy 目标**：

```
Exam07_Game（或 Exam07_GameScene）
├── Main Camera
├── Directional Light
├── EventSystem          ← 必补
└── GameCanvas           ← Prefab 实例
```

### 📌 原理加餐（A）

- **是什么**：试题7 考点在 **Game 内闭环**（商城+3D+相机）· 不是 Auth 链卷。
