## §一、考点清单（W4 阶段 1 · 对照试卷）

| 序号 | 考点 | 关键技术 | **关联考纲** | L1 验收 observable |
|------|------|----------|--------------|-------------------|
| 1 | 登录场景 UI | InputField · Button · Canvas | U01 U04 | 账号/密码框 + 登录钮可见 |
| 2 | **登录注册**逻辑 | Dictionary · Panel 切换 · Trim | U17 | 注册后可登录 |
| 3 | 登录失败反馈 | `Debug.Log` 控制台输出 | U17 | 错账号 Console 有提示 |
| 4 | 登录成功跳转 | `SceneManager.LoadScene` | U17 | 进入 Loading 场景 |
| 5 | **Loading** UI | Slider · 进度 Text | U03 U01 | 进度条 0→1 可见 |
| 6 | Loading **5 秒**完成 | 协程 · `Time.deltaTime` | U03 | **5s** 后进入 Main |
| 7 | 场景链 Build | 三场景 Index 0→1→2 | U16 | Build 顺序正确 |
| 8 | 命名 + 注释 | `SceneNames` · §11.19 | U16 | 无魔法字符串 · 可读 |

**易错点**

- Loading 用 **Exam06 `LoadingCanvas` Prefab**，禁止 UI→Canvas 手搭（O1d）  
- 登录用 **Exam01 `LoginCanvas` Prefab**，内含 `AccountInput` / `PasswordInput` / `LoginButton`  
- 协程在 **`Start` 启动**，不要只靠 `OnEnable`（L3 踩坑）  
- Log **不写明文密码**  
- Main 场景本阶段只建 **空壳**，UI 在 Stage2 搭

## §二、场景清单

| 场景文件 | 用途 | Build Index |
|----------|------|:-----------:|
| `Stage1_LoginScene.unity` | 登录/注册 | **0** |
| `Stage1_LoadingScene.unity` | 5 秒 Loading | **1** |
| `Stage2_MainScene.unity` | 主界面空壳（Stage2 再搭 UI） | **2** |

**Hierarchy 规划（Stage1 负责前两个）**

```
Stage1_LoginScene
├── Main Camera
├── EventSystem
└── LoginCanvas          ← Exam01/Prefabs/LoginCanvas · Unpack
    ├── LoginPanel
    │   ├── AccountInput
    │   ├── PasswordInput
    │   └── LoginButton
    ├── RegisterPanel（可选 · 自建或改 LoginPanel）
    └── BtnRegister / BtnToLogin（切换用）

Stage1_LoadingScene
├── Main Camera
├── EventSystem
└── LoadingCanvas        ← Exam06/Prefabs/LoadingCanvas · Unpack
    ├── ProgressSlider
    ├── TipProgressText
    └── TipTextTop/Bottom（可选）

Stage2_MainScene（空壳）
├── Main Camera
└── EventSystem
```
