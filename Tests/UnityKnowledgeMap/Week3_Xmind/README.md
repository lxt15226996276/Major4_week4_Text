```
01 角色控制器 ──┐
                ├─→ 03 状态机 ──→ 04 怪物 AI
02 摇杆插件 ────┘

05 编辑器拓展（独立，任何时候都能学）
```

|:----:|------|----------|-------------|
| 1 | 角色控制器 | Unity 基础、C# 基础 | 写玩家移动脚本，实现走跑跳 |
| 2 | 摇杆插件 | UGUI 基础、Canvas | 做手游虚拟摇杆，对接角色移动 |
| 3 | 状态机 | C# 接口基础 | 把角色行为拆成状态，告别 if-else 堆山 |
| 4 | 怪物 AI | 状态机 + 角色控制器 | 做会巡逻、追击、攻击的怪物 |
| 5 | 编辑器拓展 | C# 基础、Inspector 常用操作 | 做批量工具、自定义 Inspector、关卡编辑器 |

## 参考资料体系

每篇文档的 **参照来源** 分三层：

| 层级 | 内容 |
|------|------|
| **Unity 官方** | Manual + Scripting API（最终标准） |
| **编程专项权威** | Brackeys / GameDev.tv / Sebastian Lague / Jason Weimann 等 |
| **经典书籍** | 《游戏编程模式》《游戏 AI 编程精粹》等 |

文末均有 **「参考资料」** 表，列出本专题对应的官方手册和 API 链接。
